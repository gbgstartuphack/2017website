# Gothenburg Startup Hack
This is the official website for Gothenburg Startup Hack.

## Setup

### Prerequisites
- Node.js (v7.0)
- Gulp (installed globally)

### Install
1. Clone the repo
    `$ git clone {repo-uri}`

2. Install node dependencies
    `$ npm install`

### Running the server
- Run the server- and build process using Gulp
    `$ gulp`
